Сначала к p1 присваиваем первый элемент массива, к p2 дальше присваиваем второе минимальное значение.
Далее в цикле проходим по массиву, и сравниваем элементы. Если текущий элемент меньше p1, то обновляем значения (p2 получает значение p1, а p1 значение текущего элемента).
Если текущий элемент не меньше p1, но меньше p2 (или p2 еще nnull), то p2 обновляется на текущий элемент.
```csharp
int a = arr[i];
if (a < p1)
{
    p2 = p1;
    p1 = a;
}
else if (p2 == null || a < p2) p2 = a;
```

В итоге, если p2 равен null, то возвращается первый элемент. В противном случае сумма p1 и p2.
